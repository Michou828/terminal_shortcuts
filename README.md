# Terminal Shortcuts for Embedded Development

A centralized repository for custom terminal shortcuts, aliases, and functions, designed to streamline embedded development workflows and PlatformIO (`pio`) commands.

## Quick Install

You can install and apply these shortcuts on any new macOS or Linux machine with a single command. Open your terminal and paste:

```zsh
curl -fsSL [https://raw.githubusercontent.com/Michou828/terminal_shortcuts/main/install.sh](https://raw.githubusercontent.com/Michou828/terminal_shortcuts/main/install.sh) | zsh
